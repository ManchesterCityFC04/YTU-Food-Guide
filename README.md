## 贡献指南
如果你发现了烟大校内外好吃的美食，欢迎按照以下方式贡献内容：
1.  fork 本仓库到自己的 GitHub 账号。
2. 在对应的分类下（校内/校外），添加美食店铺/窗口的名称、详细地址、推荐菜品、个人评价等信息。
3. 提交 pull request，经审核后会合并到主分支。
